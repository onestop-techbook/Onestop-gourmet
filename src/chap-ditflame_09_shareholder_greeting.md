# 株主優待で広がる(といいつつもそんなには広がらない)食の世界

上場企業が行っている株主向けの施策として、株主優待というものがあります。この章では、そんな株主優待と食についてまとめていきます。

せっかくなのでChatGPTにも聞いてみました。

```
株主優待とは、企業が自社の株式を一定数以上保有している株主に対して、配当金とは別に贈る特典やサービスのことです。

例えば、企業の商品券、割引券、地域の特産品、自社サービスの無料利用券などが代表的です。
目的は、株主への感謝や株式を長期保有してもらうこと、企業のファンを増やすことです。
```



## そもそも株主優待を得るには？

株主優待を得るには、それぞれの上場企業が定めた株主優待の条件に達するように、企業の株式を保持する必要があります。

現在はすべて株式は電子化されていますので、株式を購入し、保持するためにはまず証券会社に証券口座を開設する事が必須となります。なので開設しましょう。

なお、国内の株式証券取引所は現在４つあります。

 * 東京証券取引所（東証）
 * 名古屋証券取引所（名証）
 * 福岡証券取引所（福証）
 * 札幌証券取引所（札証）

今回のテーマとは少し外れますが、たとえば会社によっては名証や札証にしか上場していない事があり、そういった会社の株主優待を得る（≒株式を買う）ためには、証券会社がその証券取引所の銘柄を取り扱っている必要があります。

証券会社は複数の会社に証券口座を開設することもできますが、普通にカジュアルに株式運用をする場合は特定口座を開設し、申告分離課税制度を使う事になります。この時、複数の会社の証券口座の損益を通算したくなった場合は確定申告し、総合課税での処理が必要となります（そのため、年収が高くかつ利益が出ていると逆に損するケースもある）。

そういった煩雑さを避ける観点からも、最初に開設する証券口座はそういった自分のやりたい投資に沿っているか？　という点も併せて検討しておくのがオススメです。

## 株式優待狙いでの銘柄の選び方

銘柄選びは、基本的に個別で探すか、まとまった情報を見る形になります。

### 選び方１：個別に探す

基本的には、気になる上場企業の情報を見て、株主優待情報を漁っていくのがオーソドックスな探し方です。

例えば日本経済新聞のWebには各上場企業毎のページがあり、株主優待がある銘柄の場合は「株主優待」ボタンが活性化しています。

株主優待ボタンを押した時の表示はこのようになっています。


例：ユキグニファクトリー
https://www.nikkei.com/nkd/company/?scode=1375

https://www.nikkei.com/nkd/company/yutai/?scode=1375

せっかくなので、画面の見かたについても説明しておきましょう。

たとえば現在みているユキグニファクトリー（旧：雪国まいたけ）は必須の条件があり、優待内容も記載されています。

 * 権利確定が３月
 * １単元（100株）以上保有
 * かつ6ヶ月以上継続保有
 * 同じ株主番号にて、連続して株主名簿に記載された状態の株主を対象


| 保有株式数 | 優待品内容 |
| ---- | ---- |
| 100株以上300株未満 | 3,000円相当の自社製品セット |
| 300株以上1,000株未満 | 5,000円相当の自社製品セット |
| 1,000株以上 | 7,000円相当の自社製品セット |


つまり、例えば、次のような株の持ち方ではダメということになります。

 * 3月末日までに株を売っちゃって株主ではなくなった
 * ミニ株などで100株未満の株数を持っている株主
 * 3月中旬に株を買って、新たに株主になったケースのその年の株主優待(ずっと保持していれば翌年からは貰えます)
 * 前年の9月末日に株を持っていたが、その後10月以降に全株を売っぱらって、その後3月に買い戻した(すべての株を売り切って、その後株を買うと株主番号はリセットされる為)

また、100株と200株ではもらえるものは一緒、300株から900株ではもらえるものは一緒、1000株以上はもらえるものは一緒であることがわかります。

つまり株主優待狙いだと、100株で止めるか、300株で止めるか、1000株で止めるのが一番投資としてはコスパが良くなります。

株主優待のルールは各々の会社が勝手に決めてよい(後述しますが、毎年内容が変わってもよい)ものなので、都度都度確認しておきましょう。

### 選び方２：まとまったサイトで探す

個別に株を見ていく形も楽しくて良いのですが、手際よく探したい場合には、みん株などの株主優待情報がまとまっているサイトを見るのが楽ちんです。

https://minkabu.jp/yutai/popular_ranking

なお、みん株の場合は人気ランキングの他にも、その月で優待権利が確定する銘柄の一覧などもあり、ちょっと株主優待をゲットしてみたいな～　という人がカタログショッピング的に物色するのにもうってつけです。

## 株主優待目的で投資をする前に知っておきたい前提の話

株主優待目的で株式投資を初めてみるのはおすすめなのですが、いくつか知っておく方が良い事があります。

この節ではそういった「基本的なポイント」を列挙していきます。

### 会社が株主優待をやる理由

会社が株主優待をやる理由は、表向きは「会社の製品や、会社にまつわる品を株主に使ってもらってファンとなってもらいつつ、株主に還元する」とか言われがちですが、案外そうでもないケースもあります。

というのも、株式上場を維持し続けるには、株式の流通量や時価総額、会計関連の基準だけではなく、株主数が必要となります。

この株主数というのが厄介で、単純に事業の規模が小さかったり、利益が出にくいセグメントでの事業の場合は、当然配当が少なかったり出せない　という会社もあるのですが、そういった会社は市場でも株式売買が少なくなったり、株主数も少ない　という傾向が明確にあるようです。

ところが、株主優待を始めるとあらふしぎ！　株主優待目的で買ってくれる(≒ずっと持っててくれる)個人投資家　というのが一定数いるので、公開株式がより安全に運営できるようになる　というわけです。

なお余談ですが、株主優待は個人投資家にとってはメリットですが、機関投資家からするとデメリットに近いものらしく（配当を出さずに、かつ処理の困る株主優待を送られても扱いに困るのでデメリットしかないらしい……）、会社が個人投資家に向いているか、機関投資家に向いているかで株主優待を会社として継続するかどうかの判断はガラッと変わる事があります。

### 会社が株主優待をやめるとどうなる？

たとえばJT(日本たばこ産業)だと、かつてはグループ会社の食品などを大量に株主優待で送ってくる銘柄だったのですが、これがある時を境に廃止、株主優待がなくなった事で株価は一旦大幅に下落しましたが、堅実な経営を続けて配当重視の運営に変えていったことで、現在では高い株価水準に戻っています。

株主優待をやめると、株主優待狙いの株主が一気に株を売るので、基本的には株価はガクンと下がります。

なお、大体の場合はガクンと下がった後にしか株主優待が廃止された事には気付けないものなので、焦って動いても仕方がないケースが多いです。落ち着いて適当な所で売りましょう（もし含み益出ていて、売ってもいいかなという銘柄があれば、それもセットで売って、損益通算しておくとなお良いです）。

### 会社が株主優待を新設したり、パワーアップさせるとどうなる？

逆のケースとして、株主優待の優待内容が新設されたり、元々あったものからパワーアップする事があります。

これは大体のケースで株価が跳ねます。よかったですね。

### モノを送ってくる株主優待だと、モノの内容がその会社の本業に近いかどうかを見たほうがいい

株主優待にも色々あり、モノを送ってくるタイプの株主優待だと、大きくわけると次の３つになります。

 * 本業に密接に関係した、自社製品を送ってくるもの
 * 本業に実は関係している製品（自社製品が使われている商品）を送ってくるもの
 * 本業に全然関係ないモノを送ってくるもの

この時、本業に全然関係ないモノを送ってくるケースだと「本社所在地の名産をぜひ株主の皆様に味わっていただき、地場振興したい」みたいな事が書いてあるのですが、往々にしてこういう株主優待は会社の雲行きが怪しくなると真っ先に切られがちです。

だって買ってきて送る事になるので、株主優待品の調達コスト、高いはずなのです……

### 株主優待で送られてくるものが必ず全て美味しいかと言われると

前項で「本業に全然関係ないモノ」は株主優待廃止リスクが高い　という話をしましたが、本業に関係ある製品を送ってくる会社についても、それが手放しで良いかどうかは悩ましいケースがあります。

株主優待で、新商品です！　って新商品を意気揚々と送ってくるケースはままあるのですが、これは美味しい事もあれば、全く美味しくない事もあり……　（なお火鍋スープの素が来た事があって、作ってみたら辛すぎて食べられたものではなかったｗ）

基本的には本業と、それを生かした関連製品を製造業は作りがちなのですが、関連製品は本業ではない(つまり食品開発のスキルが低いけど現場が頑張らされている)ケースもままあり、こういうのは悩ましさだよなぁ　という気がしています。

### 自社製品が使われている商品を送ってくるタイプの優待

珍しいのですがこういうケースの優待をしている会社もあります。

なお、会社によっては非常にバラエティに飛んだ品々が来るようです。

有名どころだと、大成ラミック(食品用パッケージをひたすら作ってる会社)とか面白いと思いますよ。

https://minkabu.jp/stock/4994/yutai

### モノではなく商品券が来るパターンの優待

製造業ではなく、飲食店業態での事業が主となる会社の優待の場合は、基本的には製品ではなく商品券が来るパターンの優待が多いです。

商品券が来るパターンの優待の銘柄を買う時は、商品券の有効期限と額面や内容などを見ておくのがおすすめです（使い切れないと丸々損なので）。

大体の場合は、その飲食グループの各店舗で共通で使える商品券となっている事が多いので、行きたい店舗が近くにあるならその会社の株を買ってみても良いかも知れません。

また、商品券が来るパターンの優待は、会社によっては配送ギフトに切り替えられるケースがあります（飲みメインの飲食業なんかはそういうオプションがある場合が多い印象）

### 株主優待のコスパだけを見ると、投資単位が小さいほど良い

株主優待は、ほとんどの会社で必要な株数が指数的に増えていくのに対し、優待内容は線形的にしか増えない設計になっている事が多いです。

加えて前述の通り、株主優待を実施する最大の目的はずっと株を保持していてくれる株主の確保なので、会社としては最低単元株数だけ持っていてくれさえすればよく、更にそういう会社ほど配当に関してはしょっぱい　という傾向が実際あります。

なので優待目的で株を買うなら、株主優待が得られる範囲で、なおかつできるだけ下限ギリギリを狙うのがコスパ観点からは明らかにおすすめです。

### 1年以上持っていないとダメとか、長期保有が必須な優待をうまく得たい場合

実際に色々みていくとわかるのですが、実は継続的に銘柄を持っていると、優待内容がどんどんパワーアップするタイプの優待があります。
この「継続的に持っている」という条件は株主番号でみているので、単元株(一般的には100株)を持っていれば何株持っていても「持っている」状態を維持できます。

具体例をあげると、例えば狙いの優待が「3年以上保持＆1000株以上」という条件だった場合、

 * 3年前に最低単元株(売買できる最低単位が100株と仮定)で100株買う
 * 今年の権利確定前に900株買う

という形でも「3年以上保持」しており、「1000株以上保持」している事になります（なれます）

※なおマイナンバーが証券口座登録に必須になった関係で、マイナンバーで株主の名寄せがされるようになった様。具体的には、例えば複数の証券口座で同一銘柄の株式を分散して保持していても、おなじ株主番号がどうやら維持されているような動きに手元ではなっていました(昔は引き継がれなくって、大層難儀したのですが……)。

ルールを上手に把握して、手元資金を極力ロックインしない形でうまく運用していきましょう。
